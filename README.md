# Billion Meta Lab 常用字体

为了使大家有更好的字幕体验，我们组只会使用一些免费、体积小、常用、覆盖面全（简繁）的字体制作字幕。

我们尽量保证每一部番对应一个中文字幕、一个日文字幕（日文字幕用于制作OP和ED，若无日字则会省去）

同时，我会在这里同步一定的字体。

我们的字体不会有特别多的变化，一般会使用已经使用过的字体。

就是这样。
